Different Filtering techniques

1> Using Where-Object
   The where-Object cmdlets filters objects based on specified
   criteria.
Code |
Get-Process | Where-Object{$_.CPU -gt 10}

This command retrieves all processes and filters them,
displaying only those wuth CPU usage greater than 10.

2> Filtering with Cmdlet Parameters : 
Many cmdlets have built-in paramters for Filtering
more efficient than where - object

Get-Service -Name "Win*"

This command retrieves services whose names start
with "Win".
